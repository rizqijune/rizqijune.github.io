Cara Menyelamatkan Data Android Jika Layar Pecah

Masalah

Kejadian tidak terduga sering terjadi dengan smartphone kita. Entah itu kerusakan karena umur atau kerusakan karena kelalaian kita sendiri. Kejadian seperti itu sangatlah sulit untuk diprediksi dan jika sudah terlanjur terjadi pasti anda akan kehilangan data penting yang berada di smartphone anda.

Karena masalah itu sebaiknya anda mengurangi resiko kehilangan data dengan mengatur
Hal yang harus dilakukan saat pertama kali mempunyai smartphone

Solusi

Pastikan anda sudah melakukan semua seperti yang dijelaskan pada Hal yang harus dilakukan saat pertama kali mempunyai smartphone. Jika belum, maka hal satu-satunya yang bisa anda lakukakan hanyalah datang ke service center terdekat.

Proses

Saya anggap kamu sudah melakukan langkah-langkah yang disebutkan di section solusi. Oke mari kita lanjutkan :

Perlu ROOT atau Temp ROOT

  1. USB Debugging sudah dinyalakan.
  2. Install Driver sesuai Vendor dan model Smartphone kamu.
  3. Install ADB dan Fastboot di PC kamu. Disini saya menggunakan UBUNTU.
  4. Sambungkan Smartphone ke PC kamu.
  5. Di UBUNTU, Pergi ke /home/username pc kamu/.android/ untuk Windows, pergi ke C:\Users\username pc kamu\.android\
  6. Ubah nama file adbkey.pub menjadi adb_keys
  7. Pindahkan file kemanapun di Smartphone kamu.
  8. Gunakan File explorer apapun yang bisa mengakses Directory Root.
  9. Pindahkan adb_keys ke /data/misc/adb/
  10. Reboot Smartphone kamu OPTIONAL
  11. Buka Terminal atau CMD dan ketik adb devices

Perlu disambungkan ke komputer sebelumnya

Perhatian Cara ini hanyalah untuk mengaktifkan Keys pada PC baru yang sebelumnya belum pernah disambungkan ke Smartphone atau tidak muncul Prompt dialog ADB di Smartphone anda.

  1. USB Debugging sudah dinyalakan.
  2. Install Driver sesuai Vendor dan model Smartphone kamu.
  3. Install ADB dan Fastboot di PC kamu. Disini saya menggunakan UBUNTU.
  4. Sambungkan Smartphone ke PC kamu.
  5. Di UBUNTU, Pergi ke /home/username pc kamu/.android/ untuk Windows, pergi ke C:\Users\username pc kamu\.android\
  6. Buka Terminal untuk UBUNTU atau CMD untuk Windows, arahkan pada folder diatas atau tekan F4 untuk Dolphin dan tekan Shift+Klik Kanan untuk Windows dan ketik adb devices
  7. Ketik adb push adb_keys /data/misc/adb
  8. Reboot.
  9. Voila! It works!

Sekarang yang perlu kamu lakukan hanya menginstall SCRCPY di pc kamu dan salin data penting kamu!

Kesimpulan

Begitulah cara untuk menyelamatkan data penting saat kamu lupa untuk mencadangkannya di Cloud Storage atau Cold Storage. Namun cara diatas hanya berlaku untuk smartphone yang sudah diROOT. Itulah kenapa memiliki smartphone sepenuhnya itu wajib. Tanpa ROOT, anda belum memiliki smartphone anda sepenuhnya.

Keuntungan

Dengan adanya ROOT dan persiapan yang matang sebelum hal buruk terjadi. Sebaiknya antisipasi dengan berbagai cara.
